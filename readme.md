# Proyecto final del modulo 1 de dev-f
## Proposito del proyecto.
Este proyecto es un sitio web sobre la película de CHAINSAW MAN - EL ARCO DE REZE.

Es una landing page sobre la película, dividida en varias secciones como:
- Sinopsis: Ofrece una descripción general de la película.
- Personajes: Divida en dos secciones, ofrece una descripción de algunos personaje.
- Trailer: Da la oportunidad de ver el trailer desde el sitio.
- Critica: Se pueden visualizar los resultados optenidos por la película.

## Tecnologías usadas en el proyecto.
- HTML.
- CSS.
- JS.
- Tailwind.
- AnimateCss.

## Como ver el sitio web.
URL pública del sitio: ``https://thejosefer10.github.io/Chainsaw-Man-Arco-de-Reze/``
